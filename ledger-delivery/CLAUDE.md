# Ledger Delivery Pipeline

This is the Ledger delivery pipeline — takes verified insights, calls Sisi's video service, delivers via Photon Spectrum iMessage.

### Credentials

All credentials live in the repo-root `.env` (one level up). Never commit. Never log values.

- `PHOTON_WORKSPACE_ID`
- `PHOTON_PROJECT_ID`
- `PHOTON_API_KEY`
